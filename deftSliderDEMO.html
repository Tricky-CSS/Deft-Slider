Declaration & Implementation

For using Deft-slider in your page you should connect two files to your site. JS part contain core for autonomous employment of functionality. CSS part contain basic styles for main elements of module structure.
Follow this steps to add Deft-slider to your site/page/blog:

  1. Download all this files (find it at the bottom of page in "Files" block).
  2. Extract archieve and arrange each file in typical folder on your site host (JS -> into folder with JavaScript files and CSS -> on folder with stylesheets).
  3. Append to each page, when will be used Deft-slider, next code before closing </head> tag.
    <link rel="stylesheet" href="---your site folder ---/css/slider.css" type="text/css">
    <script src="---your site folder ---/js/slider.js" type="text/javascript"></script>

  4. Plain your sliders content into simple structure and place it in code:

    <div id="yourSliderId" title="700" class="deftSlider">
      <div> --- PLACE YOUR BLOCK HERE --- </div>
      <div> --- PLACE YOUR NEXT BLOCK --- </div>
      <div> --- OR ANOTHER BLOCK HERE --- </div>
    </div>

Every block, marked as "deftSlider" class, will be matched as element of "Deft Slider" class and will be transformed for correct functionality of module. Be carefull, and check your other CSS classes for coincidence. In header of slider declaration you can use special map of attributes for customize your slider, when:
id - required tag, which identy your slider with personal settings;
title - required tag, whiht set time interval in milliseconds of slide time;
class="deftSlider" - requirred tag, which identy your constructions as "Deft Slider" element.

Every child element of "deftSlider" class perceive as separate container, when you will place your slide data. Be carefull and don't forget to close every container by closing tag with "/".
Slide can work with different children structures, but better way - place every child in <div> container.
